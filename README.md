# tools

## 项目目录结构

````
tools/
├── Apps/
│   ├── Common/
│   ├── Dev/
│   ├── Home/
│   ├── Oline/
│   └── WebMaster/
├── Runtime/
├── static/
│   ├── img/
│   ├── css/
│   ├── js/
│   └── fonts/
├── index.php
└── README.md

````

本应用基于 ThinkPHP，`Apps` 为应用根目录，目录分别包含 `Common` 公共函数目录和 `Dev`、`Home`、`Oline`、`WebMaster`四个模块目录组成。`Rumtime` 为运行时缓存和临时文件保存目录。`static` 为静态资源存放目录。`index.php` 为应用程序入口文件。

## 分类目录设想

* 首页

* 站长工具
    * Alexa 查询
    * 域名 Whois 查询
    * Whois 历史查询
    * IPC 备案查询
    * RP 查询
    * ip 地址查询


* 开发者工具
    * 字符串加密解密
    * 代码格式化
    * 正则表达式测试


* 在线查询
    * 身份证查询
